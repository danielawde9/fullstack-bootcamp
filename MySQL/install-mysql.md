 **Summary** : in this tutorial, you will learn step by step how to install
MySQL on the Windows platform using the MySQL Installer. After the tutorial,
you will have a MySQL server and its tools up and running on your system for
learning and practicing.



Note that if you use other operating system platforms such as Linux and macOS,
you use the following tutorial:



If you want to install MySQL on the Windows environment, using MySQL installer
is the easiest way. MySQL installer provides you with an easy-to-use wizard
that helps you to install MySQL with the following components:



To download MySQL installer, go to the following link
<http://dev.mysql.com/downloads/installer/>. There are two installer files:



To install MySQL using the MySQL installer, double-click on the MySQL
installer file and follow the steps below:
